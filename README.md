# L’Atelier M.S — Lovable V9

Version React/Vite moderne, pensée pour convertir les visiteurs en appels, visites et demandes de devis.

## Ouvrir sur Mac

1. Dézipper le dossier.
2. Ouvrir le dossier dans VS Code.
3. Ouvrir le terminal dans VS Code.
4. Lancer :

```bash
npm install
npm run dev
```

5. Ouvrir le lien affiché, généralement :

```txt
http://localhost:5173
```

## Mettre sur Vercel

1. Envoyer tout le dossier sur GitHub.
2. Importer le projet sur Vercel.
3. Vercel détectera Vite automatiquement.
4. Cliquer sur Deploy.

## Commandes utiles

```bash
npm run dev
npm run build
npm run preview
```
